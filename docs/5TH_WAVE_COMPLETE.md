## 🚀 Quick Start

### 1. Install Dependencies (2 minutes)
```bash
./install-5th-wave.sh
```

### 2. Start Backend (30 seconds)
```bash
cd comprehensive_backend
python main.py
```

### 3. Start Frontend (30 seconds)
```bash
cd comprehensive_frontend
npm run dev
```

### 4. Access Enhanced AI (instant)
```
http://localhost:3000/ai-predictions
```

python comprehensive_backend/test_enhanced_ai.py
